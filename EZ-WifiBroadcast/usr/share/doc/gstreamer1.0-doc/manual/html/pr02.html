Who should read this manual?

This book is about GStreamer from an application developer's point of view; it describes how to write a GStreamer application using the GStreamer libraries and tools. For an explanation about writing plugins, we suggest the Plugin Writers Guide.

Also check out the other documentation available on the GStreamer web site.
